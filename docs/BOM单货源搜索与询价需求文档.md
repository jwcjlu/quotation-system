# BOM 单货源搜索与询价需求文档

基于多平台 BOM 配单、货源搜索与价格比对的功能需求。

---

## 1. 系统概述

本系统为电子元器件采购的 **BOM 单货源搜索与询价** 功能模块，支持用户导入 BOM 文件，自动解析物料清单，在多平台（立创商城、云汉芯城、ICGOO 等）上搜索报价，并根据搜索结果自动完成配单与价格比对。

**核心流程：** 导入 BOM → 解析 BOM → 多平台搜索报价 → 自动配单

---

## 2. 功能模块

### 2.1 BOM 单导入功能

**功能描述：**
- 提供 BOM 文件导入入口，支持用户上传 BOM 单
- 支持 Excel 格式（.xlsx、.xls）

**核心功能点：**

| 功能项 | 说明 |
|--------|------|
| Excel 拖拽上传 | 支持将 Excel 文件拖拽至指定区域上传 |
| Excel 文件上传 | 点击「Excel文件上传」按钮选择本地文件 |
| 解析模式选择 | 支持选择预设解析模式或自定义列映射，见 2.2 |
| 模板下载 | 提供标准 BOM 模板，提升解析与配单准确率 |
| 历史 BOM 查看 | 查看历史配单记录 |

**界面结构：**
- 中央上传区：Excel 拖拽区域 + 上传按钮
- 底部：模板下载链接、历史 BOM 链接

---

### 2.2 BOM 单解析

**功能描述：**
- 用户导入 BOM 单后，系统按所选解析模式解析文件内容
- 从 BOM 中提取型号、厂牌、封装、数量、参数等字段

**解析模式选择：**

用户在上传 Excel 前或上传后，可选择解析模式，以适配不同来源的 BOM 格式：

| 解析模式 | 说明 | 适用场景 |
|----------|------|----------|
| 立创标准 | 按立创商城 BOM 模板列顺序解析 | 从立创导出的 BOM |
| 云汉标准 | 按云汉芯城 BOM 模板列顺序解析 | 从云汉导出的 BOM |
| 通用模式 | 自动识别表头，按列名匹配（型号、厂牌、封装等） | 常见 BOM 格式 |
| 自定义映射 | 用户手动指定：型号→A列、厂牌→B列、封装→C列… | 非标准格式 |

**自定义映射界面：**
- 展示 Excel 首行预览，供用户确认表头
- 下拉选择各字段对应的列（A/B/C… 或列名）
- 支持保存为自定义模板，下次复用

**解析规则：**

| 解析项 | 说明 |
|--------|------|
| 型号 | 从 BOM 中识别元器件型号（如 MP1658GTF-Z） |
| 厂牌 | 识别制造商/品牌（如 MPS、TI） |
| 封装 | 识别封装类型（如 TSSOP-16、LGA-16） |
| 数量 | 识别需求量 |
| 参数 | 其他技术参数（电压、电流等） |

**输出：**
- 解析后的结构化物料列表，供后续多平台搜索使用
- 支持用户对解析结果进行编辑、修正

---

### 2.3 多平台货源搜索

**功能描述：**
- 根据解析到的 BOM 单，依次在以下平台搜索报价
- 支持多平台并行或串行搜索，汇总各平台报价数据

**平台列表：**

| 平台 | 网址 | 说明 |
|------|------|------|
| 立创商城 | https://www.szlcsc.com/ | 电子元器件商城 |
| ICGOO | https://www.icgoo.net/ | 电子元器件采购平台 |
| 云汉芯城 | https://www.ickey.cn/ | 搜索入口：search.ickey.cn |

**搜索流程：**
1. 遍历解析后的 BOM 物料列表
2. 对每个物料型号，依次在各平台搜索
3. 采集各平台返回的：型号、厂牌、封装、库存、货期、价格梯度、单价等
4. 汇总各平台报价数据，供自动配单使用

**匹配与选型规则：**

各平台搜索后通常返回多条记录，系统需按以下规则筛选并选择一条作为推荐：

| 规则 | 说明 |
|------|------|
| 型号匹配 | 平台返回的型号与 BOM 解析出的型号一致 |
| 封装匹配 | 平台返回的封装与 BOM 解析出的封装一致 |
| 厂牌匹配 | 平台返回的厂牌与 BOM 解析出的厂牌一致 |
| 价格最低 | 在满足上述三项匹配的记录中，选择单价最低的一条 |

若某平台无完全匹配记录，则该平台不参与该物料的推荐；若多平台均有匹配，则跨平台比较后取价格最低者。

**数据采集字段：**

| 字段 | 说明 |
|------|------|
| 型号 | 匹配到的型号 |
| 厂牌 | 供应商/制造商 |
| 封装 | 封装类型 |
| 库存 | 可用库存 |
| 货期 | 交货周期 |
| 起订量 | 最小起订量 |
| 价格梯度 | 阶梯价格（如 5+ ￥0.88 \| 50+ ￥0.74） |
| 中国香港交货 | 香港交货价格 |
| 内地交货(含增值税) | 内地含税价格 |

### 2.4 自动配单

**功能描述：**
- 根据多平台搜索报价数据，自动完成配单
- 各平台返回多条记录时，按 **型号、封装、厂牌均匹配** 且 **价格最低** 的规则选型（详见 2.3 匹配与选型规则）
- 支持按价格、库存、货期等策略选择最优货源
- 支持多平台货源混选（同一 BOM 中不同物料可从不同平台采购）

**配单策略（可配置）：**

| 策略 | 说明 |
|------|------|
| 价格优先 | 选择单价最低的货源 |
| 库存优先 | 选择库存充足的货源 |
| 货期优先 | 选择货期最短的货源 |
| 综合排序 | 综合价格、库存、货期等维度 |

**输出：**
- 每行物料对应推荐最优型号及来源平台
- 库存、货期、单价、小计
- 支持用户手动调整或确认

---

### 2.5 BOM 配单结果与价格比对

**功能描述：**
- 展示 BOM 匹配结果，支持按匹配状态筛选
- 每行默认显示系统推荐的最优匹配；支持「显示更多」展开，查看该物料在各平台搜索到的全部报价
- 展示每行物料的库存、货期、单价、小计

**核心功能点：**

#### 2.5.1 匹配状态筛选

| 状态 | 说明 |
|------|------|
| 全部 | 显示所有匹配结果（示例：26 条） |
| 完全匹配 | 系统自动匹配成功的型号（示例：20 条） |
| 待确认 | 需人工确认的推荐型号（示例：4 条） |
| 无法匹配 | 匹配失败的型号（示例：2 条） |

#### 2.5.2 表格列定义

| 列名 | 类型 | 说明 |
|------|------|------|
| 序号 | 数字 | 行号 |
| 原始需求 | 文本 | 用户上传的原始数据（如：MP1658GTF-Z MPS, 4804...） |
| 整理后需求 | 结构化 | 解析后的需求：型号、厂牌、封装、参数；支持编辑、删除 |
| 总需求量 | 数字 | 用户需求数量 |
| 结果 | 状态图标 | 红色 X：无法匹配；橙色 !：待确认；绿色 ✓：完全匹配 |
| 推荐最优型号 | 结构化 | 默认显示系统推荐的最优匹配；支持「显示更多」展开，展示该物料在各平台搜索到的全部报价结果 |
| 库存/货期/批次 | 结构化 | 库存数量、货期（如 5-7工作日）、批次 |
| 购买数量 | 输入框 | 支持步进；显示起订量、增量、整包数 |
| 单价(含税) | 金额 | 含税单价（如 ¥0.92） |
| 小计 | 金额 | 该行总价（自动计算） |
| 操作 | 按钮 | 加入购物车、更换型号、确认型号、**查看详情** |

#### 2.5.3 查看详情

**功能描述：**
- 每行被匹配的记录项增加「查看详情」按钮
- 点击后弹出/展开详情面板，展示该物料在**各平台搜索到的全部报价记录**
- 详情内容包含：平台名称、型号、厂牌、封装、库存、货期、价格梯度、单价等，便于用户对比各平台报价并手动更换选择

| 展示项 | 说明 |
|--------|------|
| 平台 | 立创商城、云汉芯城、ICGOO 等 |
| 各平台记录列表 | 该平台搜索到的所有匹配记录（含未选中的） |
| 当前选中 | 高亮或标记当前推荐的最优记录 |

#### 2.5.4 其他功能

| 功能 | 说明 |
|------|------|
| 查看重复物料 | 复选框，控制是否显示重复物料 |
| 显示配置 | 下拉菜单，控制表格列显示 |
| 价格波动提示 | 黄色提示：价格/库存可能波动，以结算为准 |
| 显示更多 | 每行「推荐最优型号」支持展开，点击后展示该物料在各平台搜索到的全部报价（型号、平台、库存、货期、单价等），便于对比与更换 |

## 4. 业务规则

1. **BOM 导入**：用户导入 BOM 后自动触发解析，解析失败时提示用户检查格式或下载模板
2. **多平台搜索**：按 BOM 解析结果依次在各平台搜索，支持多平台并行以提升效率
3. **价格与库存**：以结算页为准，配单页展示价格可能波动
4. **匹配逻辑**：优先完全匹配（型号、封装、厂牌均一致），在匹配记录中选价格最低者；其次推荐待确认；无法匹配时提示用户修改
5. **购买数量**：需满足起订量，按增量单位调整

```
       │                    │                      │                      │
       │                    │                      │                      │
  用户上传 Excel        提取型号/厂牌/        立创商城、ICGOO、        按策略选最优

```

## 7. 与现有爬虫的关联

| 平台 | 现有实现 | 说明 |
|------|----------|------|
| 云汉芯城 | `ickey_crawler.py` | 已支持 search.ickey.cn 单型号搜索，可扩展为 BOM 批量查询 |
| 立创商城 | 待开发 | 需新增 szlcsc.com 搜索与解析逻辑 |
| ICGOO | 待开发 | 需新增 icgoo.net 搜索与解析逻辑 |

**扩展建议：**
1. **统一爬虫接口**：抽象各平台为统一接口（搜索、解析），便于接入新平台
2. **批量查询**：在现有单型号爬虫基础上，支持 BOM 列表批量调用
3. **数据标准化**：各平台返回数据统一映射为：型号、厂牌、封装、库存、货期、价格梯度、单价等
