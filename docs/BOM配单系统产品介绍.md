# 1. 产品概述

**BOM 配单系统** 是一款面向电子元器件采购的智能询价与配单平台。

**核心能力：**
- 导入 BOM 清单，自动解析物料
- 多平台并行搜索报价（云汉芯城等）
- 智能匹配选型，推荐最优货源
- 一键查看详情，跨平台比价

---

# 解决的核心痛点

| 传统方式 | 本系统 |
|----------|--------|
| 手动在各平台逐个搜索型号 | 一键批量搜索，多平台并行 |
| 人工比对价格、库存、货期 | 自动匹配选型，推荐最优 |
| Excel 格式不统一，解析困难 | 通用/自定义解析，灵活适配 |
| 难以追溯历史配单 | 历史 BOM 可查看、可复用 |

---

# 2. 核心价值

- **提效**：从数小时缩短至分钟级完成 BOM 询价
- **降本**：跨平台比价，自动选价格最低的完全匹配货源
- **准确**：型号、封装、厂牌三重匹配，减少错配风险
- **灵活**：支持通用模式与自定义列映射，适配各类 BOM 格式

---

# 3. 功能模块概览

```
┌─────────────┐    ┌─────────────┐    ┌─────────────────────┐    ┌─────────────┐
│ BOM 导入    │ -> │ BOM 解析    │ -> │ 多平台搜索报价      │ -> │ 自动配单    │
└─────────────┘    └─────────────┘    └─────────────────────┘    └─────────────┘
  拖拽/上传          提取型号/厂牌        云汉芯城等平台           智能选型推荐
  模板下载          封装/数量            并行搜索                查看详情比价
```

---

# 3.1 BOM 导入与解析

**支持格式：** Excel（.xlsx、.xls）

**解析模式：**

| 模式 | 说明 |
|------|------|
| 通用模式 | 自动识别表头，按列名匹配 |
| 自定义映射 | 手动指定型号→A列、厂牌→B列等 |

**提取字段：** 型号、厂牌、封装、数量、参数

---

# 3.2 多平台货源搜索

**已接入平台：**
- 云汉芯城（search.ickey.cn）

**搜索能力：**
- 批量型号并行搜索
- 采集：型号、厂牌、封装、库存、货期、价格梯度、单价

**数据字段：** 库存、货期、起订量、阶梯价格、香港/内地交货价

---

# 3.3 智能匹配与选型

**选型规则：**

1. **完全匹配筛选**：型号、封装、厂牌三项均一致
2. **价格最低选型**：在完全匹配记录中取单价最低

**输出：**
- 每行物料推荐最优型号及来源平台
- 库存、货期、单价、小计

---

# 3.4 配单结果与详情

**表格展示：**
- 序号、原始需求、整理后需求、总需求量
- 匹配状态（完全匹配 / 待确认 / 无法匹配）
- 推荐最优型号、库存/货期、购买数量、单价、小计

**查看详情：**
- 点击「查看详情」展示该物料在各平台的全部报价
- 按平台分组，高亮当前选中记录，便于对比更换

---

# 4. 业务流程

```
用户上传 Excel
      │
      ▼
解析 BOM（通用/自定义）
      │
      ▼
提取型号、厂牌、封装、数量
      │
      ▼
多平台并行搜索报价
      │
      ▼
按规则筛选完全匹配 + 价格最低选型
      │
      ▼
生成配单结果，支持查看详情
```

# 5. 技术优势

| 维度 | 说明 |
|------|------|
| **架构** | Go + Kratos 微服务框架，高并发、易扩展 |
| **解析** | 通用模式自动识别 + 自定义列映射，适配各类 BOM |
| **搜索** | 平台抽象层，便于接入新平台 |
| **匹配** | 型号/封装/厂牌三重校验，价格优先选型 |

---

# 6. 应用场景

- **研发采购**：新项目 BOM 批量询价，快速获取各平台报价
- **成本优化**：跨平台比价，自动推荐价格最低的完全匹配货源
- **采购比选**：查看详情对比各平台报价，支持手动更换选型
- **历史追溯**：历史 BOM 可查看，便于复盘与复用

---

# 典型使用流程

1. **上传 BOM**：拖拽或选择 Excel 文件，选择解析模式（通用/自定义）
2. **确认解析**：查看解析后的物料列表，必要时编辑修正
3. **触发搜索**：系统自动在各平台搜索报价
4. **查看配单**：按匹配状态筛选，查看推荐结果
5. **查看详情**：点击「查看详情」对比各平台全部报价，按需更换

**BOM 配单系统** 帮助电子元器件采购人员：

- 从 **人工比价选型** 到 **智能匹配推荐**
- 从 **格式不统一** 到 **灵活解析适配**

**提升效率 · 降低成本 · 减少错配**
