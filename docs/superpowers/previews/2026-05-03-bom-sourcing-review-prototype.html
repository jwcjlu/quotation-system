厂牌档位与 LLM 第二意见

Screen H · 规格 §7.1

主路径（确定性）：词表 + 规则归一得到 mfr_norm。BOM 行指定厂牌时，报价归一后与行要求不一致 → 不进默认可推荐池（仍可在实验室全量中查看）。
LLM：仅在厂牌空、清洗失败、最低价但疑似错配等条件触发；结果写入 llm_brand_suggestion。合并：仅规则高置信，或模型高置信且与规则一致时，才自动上调「厂牌可信度档位」并参与默认推荐；否则 待人工 或排除自动推荐。留痕：见下表字段。

档位示意（采购一眼）

档位 条件（简化） 默认可推荐池
A 规则归一高置信，且与 BOM 厂牌一致（或未指定厂牌且规则可比对） 可参与 · 可一键确认（若其他硬约束通过）
B 规则中置信；或 LLM 高置信且 与规则 mfr_norm 一致 可参与排序 · 建议抽检 / 进队列由策略定
C 厂牌空/清洗失败；或 LLM 与规则冲突；或 BOM 指定厂牌与报价不一致 排除自动推荐 · 须人工点选或纠错后重算
