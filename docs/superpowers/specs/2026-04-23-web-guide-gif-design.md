# Web 使用指南 GIF 设计

## 目标

为前端 `使用指南` 页面补充一张循环播放的 GIF，帮助新手快速理解两段连续操作：

1. 新建 BOM 单并进入上传弹窗
2. 从会话列表打开会话看板查看数据

## 方案

- 在 `使用指南` 页中插入一张合并 GIF，而不是拆成两张
- GIF 时长控制在 10 到 16 秒之间
- 内容按以下顺序展示：
  - 首页进入 `BOM 会话`
  - 点击 `新建 BOM 单`
  - 停留展示上传弹窗关键区域
  - 回到会话列表
  - 点击 `打开详情`
  - 停留展示会话看板中的导入进度、平台勾选和 BOM 行
- 使用真实页面截图或序列帧生成，不使用虚构 UI

## 页面落点

- 将 GIF 放在 `使用指南` 页第 1 步和第 2 步附近
- 保持现有“3 分钟快速上手”结构不变
- GIF 下方保留简短说明，告诉用户它演示的是“上传 + 看板查看”完整链路

## 非目标

- 不录制管理员功能
- 不在 GIF 中演示匹配单流程
- 不制作带音频视频，仅输出前端可直接使用的 GIF 资源

## 验证

- 本地生成 GIF 文件
- 在 `使用指南` 页面中实际渲染该 GIF
- 运行前端相关测试
- 运行前端构建，确认资源能被打包
